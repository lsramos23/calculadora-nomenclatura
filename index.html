Calculadora de Nomenclatura

Use esta calculadora para definir corretamente a nomenclatura das demandas.
Selecione os tipos de criativos desejados, informe as quantidades
e adicione os formatos necessários. O sistema calculará automaticamente
a nomenclatura no formato [Número de Criativos x Número de Desdobramentos].
Clique no botão "Copiar Resultado" para facilitar o uso.
